    # 二、CSS的应用

    # 1、CSS的概念及作用

    # CSS概念：层叠样式表

    # CSS作用：精确控制网页中的元素（文字、图片、音视频，超链接等）位置，颜色，大小，布局等。

    # 2、CSS的引用方式

    # 方式1：行内样式:写在HTML的开始标签中，只对当前标签起作用

    # 方式2：内嵌样式：在head标签中，当前页面中都可以用

    # 方式3：外链样式，写在外部css文件中，当前项目中的所有页面都可以用

    先在项目文件夹中创建名为：css的子文件夹

    【Ctrl+N】,【Ctrl+S】,命名为：style.css，保存在css文件夹中

    三种引入方式的区别：应用范围不同

    # 4、background:背景

    background:red url('img/1.jpg') no-repeat 10px 20px;

    /* 颜色 背景图片路径 是否重复 水平位置 垂直位置 */

    说明：是否重复：repeat:默认的，x与y都重复

    ​ repeat-x:x方向重复

    ​ repeat-y:y方向重复

    no-repeat:不重复

    ​ 位置： 第一个表示水平（left/center/right），

    第二个表示垂直(top/center/bottom)

    也可用 10px 20px

    背景中的各个属性与顺序无关。

    Img****插入图片和backgroud中添加背景图的区别

    A、img可以右击另存为图片，background不能

    B、Img可撑开盒子，background不能

    C、Img中可以添加alt属性，添加关键字，background不能

    D、Background可以设置图片重复。

    重点：

    Padding****与margin的区别：

    当有边框时，margin会产生边框以外的空白，padding会产生边框以内的空白。

    无边框时,padding和margin会相同的空白效果，但有背景时，背景会在padding产生的空白处显示，margin不会。
